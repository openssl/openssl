=head1 NAME

X509_ACERT_EXT_add_conf,
X509_ACERT_EXT_add_nconf
- Add extensions to X509_ACERT from configuration section

=head1 SYNOPSIS

 #include <openssl/x509_acert.h>
 int X509_ACERT_EXT_add_nconf(CONF *conf, X509V3_CTX *ctx, const char *section,
                              X509_ACERT *acert);
 int X509_ACERT_EXT_add_conf(LHASH_OF(CONF_VALUE) *conf, X509V3_CTX *ctx,
                             const char *section, X509_ACERT *acert);

=head1 DESCRIPTION

B<X509_ACERT_EXT_add_nconf> enables the addition of X.509v3 extensions from an
B<CONF>, whereas B<X509_ACERT_EXT_add_conf> does the same, but from a
B<LHASH_OF(CONF_VALUE)>.

=head1 RETURN VALUES

These functions return 1 on success and 0 on failure.

=head1 HISTORY

X509_ACERT_EXT_add_nconf() and X509_ACERT_EXT_add_conf() were added in
OpenSSL 3.5.
