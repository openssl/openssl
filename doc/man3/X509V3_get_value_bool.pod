=head1 NAME

X509V3_get_value_bool, X509V3_get_value_int - parse CONF_VALUE as boolean
or integer

=head1 SYNOPSIS

 #include <openssl/x509v3.h>

 int X509V3_get_value_bool(const CONF_VALUE *value, int *asn1_bool);
 int X509V3_get_value_int(const CONF_VALUE *value, ASN1_INTEGER **aint);

=head1 DESCRIPTION

These functions are utility functions for parsing B<CONF_VALUE> structures
in X509 v3 extension B<v2i> methods (see L<X509V3_EXT_add(3)>).

X509V3_get_value_bool() parses the I<value> member of I<value> as a boolean
and sets I<*asn1_bool> accordingly. The following strings are accepted
(case-sensitive): "TRUE", "true", "Y", "y", "YES", "yes" set I<*asn1_bool>
to 0xff; "FALSE", "false", "N", "n", "NO", "no" set I<*asn1_bool> to 0.

X509V3_get_value_int() parses the I<value> member of I<value> as a decimal
integer and allocates a new B<ASN1_INTEGER> in I<*aint>. The string is
converted using L<s2i_ASN1_INTEGER(3)>.

=head1 RETURN VALUES

Both functions return 1 on success and 0 on error. On error, an appropriate
error code is pushed onto the error stack.

=head1 SEE ALSO

L<X509V3_add_value(3)>,
L<X509V3_EXT_add(3)>,
L<s2i_ASN1_IA5STRING(3)>

These functions have been available since OpenSSL 0.9.5.
