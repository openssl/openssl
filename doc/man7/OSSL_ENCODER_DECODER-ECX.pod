=head1 NAME

OSSL_ENCODER_DECODER-ECX,
OSSL_ENCODER-ED25519, OSSL_DECODER-ED25519,
OSSL_ENCODER-ED448, OSSL_DECODER-ED448,
OSSL_ENCODER-X25519, OSSL_DECODER-X25519,
OSSL_ENCODER-X448, OSSL_DECODER-X448
- ECX (X25519, X448, ED25519, ED448) encoder and decoder

=head1 DESCRIPTION

The ECX encoder and decoder implementations cover the X25519, X448, ED25519,
and ED448 key types, and are available in the default and base providers.

=head2 Encoding

The following output types are supported:

=over 4

=item text

Human-readable text representation of the key fields.

=item DER

Binary DER encoding.

=item PEM

PEM (base64-encoded DER with header/footer) encoding.

=back

The following output structures are supported:

=over 4

=item PrivateKeyInfo (PKCS#8)

Standard PKCS#8 unencrypted private key format.

=item EncryptedPrivateKeyInfo (PKCS#8)

Standard PKCS#8 encrypted private key format.
Requires a cipher to be specified via the B<cipher> parameter.

=item SubjectPublicKeyInfo

Standard X.509 public key format.

=back

ECX key types do not have type-specific output structures.  Only PKCS#8 and
SubjectPublicKeyInfo formats are supported.

=head2 Decoding

The following input types are supported:

=over 4

=item DER

Binary DER input.

=item PEM

PEM input (automatically handled via the general PEM-to-DER decoder).

=back

The following input structures are supported for DER decoding:

=over 4

=item PrivateKeyInfo

=item SubjectPublicKeyInfo

=back

=head2 Encoder Parameters

=over 4

=item "cipher" (B<OSSL_ENCODER_PARAM_CIPHER>) <UTF8 string>

The name of the cipher to use when encrypting the private key in
EncryptedPrivateKeyInfo format.  If not specified, no encryption is performed.

=item "properties" (B<OSSL_ENCODER_PARAM_PROPERTIES>) <UTF8 string>

Property query string used when fetching the cipher.

=back
