=head1 NAME

OSSL_DECODER-LMS - LMS decoder

=head1 DESCRIPTION

The LMS decoder implementation is available in the default and base providers.
LMS supports decoding only; there is no corresponding LMS encoder.

=head2 Decoding

The following input type is supported:

=over 4

=item XDR

External Data Representation format.

=back

Only public key decoding is supported.
